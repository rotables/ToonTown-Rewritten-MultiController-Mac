# ToonTown-Rewritten-MultiController-Mac
Multicontroller for ToonTown Rewritten, built to modern macOS specifications, ARM-based, and written entirely in Swift

# I know how a multicontroller works, just give me the download
You can download the app by clicking here -- soon.
**Please note that you will need to go to System Settings and allow this program to open since it is unsigned.** You can find the source code in this repository for your review.

# Why another multicontroller?
This app is written entirely in Swift and meets modern macOS specifications.

As a primary Mac player for ToonTown Rewritten, and having played personally since 2004, I recognized an opportunity to assist others in the community and continue to support a game that has played a big part in my life. Apple will be sunsetting support for the Rosetta translation layer next year, so there is a need for an up-to-date multicontroller that does not rely on Rosetta to function.

There are some new features that are based on years of using other multicontrollers.

# Features
Please note that this app was created mostly with the help of Fable 5 (Claude). There is also a disclaimer in the "about" section.

## Quickly open and use
Click the app to open, and you can click "autofill" under the windows to select to have the app autofill the TTR windows/instances. Who's been in a boss fight and realized they can't control 2 toons separately?



## Mirror and multi-toon mode, with menu bar status
Control up to 4 instances of ToonTown Rewritten. Quickly change modes by using the tilda key.




## Can be used without the window being open
You can have the multicontroller window closed and it'll still function in what ever mode you've set.

## Recognizes when you are chatting or a ToonTown Rewritten window isn't focused
The app will automatically pause inputs if you are not in a ToonTown Rewritten window or if you're typing a message. You can still send messages on all 4 toons. (disclaimer: I became friends with this toon after and explained what I was doing)



## Light-weight, entirely offline app
Uses minimal amount of RAM & CPU, with no online checks ever. I don't anticipate releasing an additional version after this unless there are changes that Apple requires to continue to function.
